Тестовый проект реализации аккордеона

Новость

Компания Intel внесла изменения в лицензионное соглашение для обновленного микрокода своих процессоров. В тексте упоминается запрет публикации результатов тестирования производительности. По новым условиям, пользователи не имеют права запускать бенчмарки, если их результаты планируется показать третьим лицам.

Изображение

Дополнительный текст

На изменения обратил внимание один из лидеров Open Source движения Брюс Перенс (Bruce Perens). Поскольку микрокод применяется при выполнении всех инструкций, правила касаются всего процессора. В то же время, запрет действует даже на тесты производительности, которые написаны самими пользователями.
